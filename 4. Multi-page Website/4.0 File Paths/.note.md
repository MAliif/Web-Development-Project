# Computer File Paths

- **Absolute file path** = relative to the root of computer. Like C:, D:  Jadi dasarnya `'/'` `[C:/home/folder]`  
- **Relative file path(short)** = lebih cepat `[../dog.jpg]`  
- **Special Characters** :  
 **'..'** = Moveback into parent directory  
 **'.'** = Current directory
